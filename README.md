# AM2 Homework

Homework solutions for the **AM2** subject (architecture and middleware).

## Assignments

| # | Folder | Topic |
|---|--------|-------|
| 1 | `01/` | Data download service — XHR vs Fetch API with progress tracking |
| 2 | `02/` | Blocking vs non-blocking CPU-bound request handling |
| 3 | `03/` | Server-Sent Events (SSE) live data streaming |
| 4 | `04/` | WebSocket proxy and chat client |
| 5 | `05/` | OAuth 2.0 browser-based app (Google, implicit flow) |
| 6 | `06/` | JWT authentication and RBAC with asymmetric keys |

## Usage

Individual projects include their own `README.md`, `package.json`, and run instructions where applicable.

```bash
cd <number>/<project>
npm install
npm start
```
